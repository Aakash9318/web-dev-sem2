# Indian Railway Dashboard - Capstone Project

## Overview
This is a modern, responsive web application for an Indian Railway Dashboard, built as a capstone project. It provides a premium, "Royal" aesthetic with features like live train status, train search, and an analytics dashboard.

## Features
- **Train Search**: Search for trains between stations with real-time status and smart predictions.
- **Live Status**: View live status of trains with a beautiful, responsive UI.
- **Analytics Dashboard**: Track seat availability trends over time using interactive charts.
- **Royal Design**: Features a premium design system with elegant typography (Playfair Display and Outfit) and a royal color palette (Deep Navy, Antique Gold, and Rich Cream).
- **Watchlist**: Add trains to a watchlist for quick access.

## Tech Stack
- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit
- **Icons**: Lucide React
- **Charts**: Chart.js and react-chartjs-2

## Running Locally

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Folder Structure
- `src/pages`: Main application views (Home, Dashboard, TrainDetails, Profile).
- `src/components`: Reusable UI components (TrainCard, Navbar, LiveStatusBadge, etc.).
- `src/store`: Redux store configuration and slices.
- `src/services`: API services and mock data.
